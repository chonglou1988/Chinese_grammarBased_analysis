# 中文语法分析项目

## 项目概述
本项目旨在分析和研究中文语法结构，为自然语言处理提供基础支持。

## 主要功能
- 中文句子结构分析
- 词性标注
- 句法树生成

## 技术栈
- Python
- NLTK
- Jieba
